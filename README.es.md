## Descripción General  
Este proyecto es una **página "Coming Soon" responsive** que se adapta a diferentes tamaños de pantalla.  
Cuenta con una sección principal (hero) con imagen, breve descripción y un formulario de suscripción por correo electrónico que incluye validación del lado del cliente.

---

## El Desafío  
Los usuarios deberían poder:  
- Ver el diseño óptimo según el tamaño de pantalla de su dispositivo.  
- Observar efectos hover en elementos interactivos.  
- Recibir un mensaje de error al enviar el formulario con un correo electrónico no válido.

## Mi Proceso  
- Estructuré el layout usando **Flexbox** y **Grid**.  
- Añadí **validación de formulario** usando JavaScript.  
- Implementé **estados hover** y **mensajes de error** para mejorar la interactividad.  
- Hice que la página fuera **responsive** en diferentes dispositivos.  

---

## Tecnologías Usadas  
- **HTML5**  
- **CSS3**  
- **Flexbox**  
- **CSS Grid**  
- **JavaScript (ES6)**  

---

## Lo Que Aprendí  
- Cómo validar entradas de formulario usando **JavaScript**.  
- Cómo usar **Grid y Flexbox** juntos para crear layouts adaptativos.  
- Gestionar **tipografía e imágenes responsivas** con CSS.  
- Crear transiciones suaves de hover y estados de error para una interfaz más pulida.
