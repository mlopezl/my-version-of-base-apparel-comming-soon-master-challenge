# Frontend Mentor - Base Apparel coming soon page solution  

This is my solution to the **Base Apparel coming soon page** challenge on [Frontend Mentor](https://www.frontendmentor.io/).  
The challenge helped me improve my **HTML, CSS, and JavaScript** skills by building a responsive landing page with form validation.

## Overview  
This project is a **responsive coming soon page** that adapts to different screen sizes.  
It features a hero section with an image, a short description, and an email subscription form that includes client-side validation.

---

## The challenge  
Users should be able to:  
- View the optimal layout depending on their device's screen size.  
- See hover states for interactive elements.  
- Receive an error message when submitting the form with an invalid email.

## My process  
- Structured the layout using **Flexbox** and **Grid**.  
- Added **form validation** using JavaScript.  
- Implemented **hover states** and **error states** for better interactivity.  
- Made the page **responsive** across different devices.

## Built with  
- **HTML5**  
- **CSS3**  
- **Flexbox**  
- **CSS Grid**  
- **JavaScript (ES6)**

## What I learned  
- How to validate form inputs using **JavaScript**.  
- How to use **Grid and Flexbox** together to create adaptive layouts.  
- Managing **responsive typography and images** with CSS.  
- Creating smooth hover and error state transitions for a polished UI.
